# Portfolio Website

A modern, responsive portfolio website showcasing projects and professional experience.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Experience Timeline**: Visual timeline showing professional experience and education
- **Projects Showcase**: Grid layout for displaying projects with tags
- **Modern UI**: Clean and professional design with smooth animations
- **Easy to Customize**: Simple HTML and CSS structure

## Sections

1. **Home**: Hero section with introduction
2. **About**: Personal introduction and background
3. **Projects**: Showcase of work and projects
4. **Exp√©rience (Timeline)**: Visual timeline of professional experience and education
5. **Contact**: Links to get in touch

## How to Use

Simply open `index.html` in a web browser to view the portfolio website.

## Customization

Edit the content in `index.html` to add your own:
- Personal information
- Projects and achievements
- Professional experience
- Contact details

Modify `styles.css` to change:
- Colors (see CSS variables in `:root`)
- Fonts
- Layout and spacing

## Technologies

- HTML5
- CSS3
- Pure JavaScript (no dependencies)
